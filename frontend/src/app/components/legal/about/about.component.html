Technical Background

Frontend

The Frontend is the here useable Graphical user interface, which is based on the software 'open-plan' from the Reiner Lemoine Institut.

Backend

The backend is the processing component of this software package.

It is composed of an API, which is the handover point between the frontend and the server. The API is based on the Python software library "FastAPI". This generates the oemof energy system from the transferred data and starts the optimization in an encapsulated Docker container.

Github

You can follow the development of this user interface, or deploy your own ensys tool (including a simulation server and user interface) by visiting our github page

Simulation server

Our simulations run on a server hosted at Hochschule Nordhausen, the code to setup the simulation server is licenced under aGPL License and publicly available on our github page. The simulation are performed with a self coded backend, available on github. A code using oemof-solph to simulate energy system model from techno-economic parameters.
